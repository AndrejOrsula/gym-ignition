gym_ignition.randomizers.physics¶

gym_ignition.randomizers.physics.dart¶

class gym_ignition.randomizers.physics.dart.DART(seed=None)¶

Bases: gym_ignition.randomizers.base.physics.PhysicsRandomizer

Class that configures the Ignition Gazebo physics with the DART physics engine and no randomization.

randomize_physics(world)¶

Method that insert and configures the physics of a world.

By default this method loads a plugin that uses DART with no randomizations. Randomizing physics engine parameters or changing physics engine backend could be done by redefining this method and passing it to GazeboRuntime.

Parameters

world (World) – A world object without physics.

Return type

None
